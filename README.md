# AI Path - Starter Repo

Files:
- hello.py - simple interactive greeting
- renamer.py - safely rename files in a folder
- count_rows.py - counts rows in a CSV file
- w1_tue_practice.py - asks you your name and age, tells you how old you will be in a year
- swordfishpassword.py - simple password algorithm which gives you 3 attempts to get the password correct

How to run:
1. Install Python 3.11 and pip
2. python hello.py
3. python renamer.py test_files
4. python count_rows.py sample.csv
5. python w1_tue_practice.py
6. python swordfishpassword.py
